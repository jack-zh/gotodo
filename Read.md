# ztodo 

A simple command-line todo list written in Go.

## Version

0.2

## Installation:

+ 1. Install Go:
  
  http://golang.org/doc/install.html

+ 2. Install todo:
  
  $ go get github.com/jack-zh/ztodo

## Usage:

    ztodo list|ls
        Show all tasks
    ztodo list|ls N
        Show task N
    ztodo rm|remove N
        Remove task N
    ztodo done N
        Done task N
    ztodo undo N
        Undo task N
    ztodo doing N
        Doing task N
    ztodo clean
        Rm done task
    ztodo clear
        Rm all task
    ztodo add ...
        Add task to list
